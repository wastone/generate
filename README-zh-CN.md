# @wastone/generate
[英文文档](README.md)|中文文档

##### 根据模板文件和数据库表生成文件
> 常用于大量重复性文件的开发需求，比如后台管理系统有很大一部分列表页面都类似，只有增删改查相关功能，可以借此库解决重复性工作，解放劳动力去研究其他专业技术。

## Install

```bash
// 适用于开发模式
$ npm install -D @wastone/generate
```

## Features

- 根据模板文件和数据库表生成文件
- 一键唤起客户端进行可视化操作
- vscode 插件
